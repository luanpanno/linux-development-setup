# Setup de desenvolvimento no Linux de Luan Panno

## O que é instalado
1. Google Chrome
2. Curl
3. Git
4. Node.js
5. VSCode
6. Extensões do VSCode
7. Spotify
8. Insomnia

## Como executar
No terminal, vá até a pasta em que o setup.sh foi baixado e execute

    chmod +x setup.sh

Em seguida, rode o setup com

    sudo ./setup.sh
